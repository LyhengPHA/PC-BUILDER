# 🖥️ PC Builder App

A mobile app for a PC shop — build custom PCs, place orders, track delivery, and chat with admin.

Built with **Flutter** + **Firebase**.

---

## Tech Stack
- Flutter 3.x (Dart)
- Firebase Auth + Firestore
- Android (iOS-ready)

---

## Getting Started

```bash
git clone https://github.com/LyhengPHA/PC-BUILDER.git
cd PC-BUILDER
flutter pub get
flutter run
```

> ⚠️ Add your own `google-services.json` and `lib/firebase_options.dart` from Firebase Console.

---

## Features

### 👤 Customer
- 🔐 Register & Login with friendly error messages
- 🛍️ Browse components by category
- 🔧 PC Builder — 7-slot selector, clears after order
- 🛒 Cart — review build, apply promo code, add notes
- 📦 Order Tracking — price breakdown + discount shown
- 💬 Real-time chat with admin
- ⚙️ Settings — edit name, change password, delete account

### 🛠️ Admin
- 📊 Dashboard overview
- 📦 Inventory — add, edit, delete, toggle stock
- 🧾 Order Management — view notes, update status, price breakdown
- 🏷️ Discount Codes — create, edit, delete with expiry dates
- 💬 Customer Messages — reply to chats in real time
- ⚙️ Settings — edit name, change password, logout
