# PC Builder App

A production-ready Flutter + Firebase Android app for custom PC building and shop management.

---

## Quick Start

### 1. Prerequisites
- Flutter 3.x — https://flutter.dev/docs/get-started/install
- Firebase CLI — `npm install -g firebase-tools`
- FlutterFire CLI — `dart pub global activate flutterfire_cli`
- Node.js (for seeding components)

### 2. Create Firebase Project
1. Go to https://console.firebase.google.com
2. Create a new project (e.g. `pc-builder-app`)
3. Enable Authentication → Email/Password
4. Enable Firestore Database (production mode)
5. Enable Storage

### 3. Connect Flutter to Firebase
```bash
cd pc_builder_app
flutter pub get
flutterfire configure
```
This generates `lib/firebase_options.dart` with your real config.

### 4. Apply Firestore Security Rules
In Firebase Console → Firestore → Rules, paste the contents of `firestore.rules`.

### 5. Seed Component Data
```bash
npm install firebase-admin
# Download serviceAccountKey.json from Firebase Console → Project Settings → Service Accounts
node seed_components.js
```

### 6. Create an Admin Account
1. Register normally in the app
2. Go to Firestore → users collection → find your doc
3. Change `role` from `"customer"` to `"admin"`
4. Log out and back in — you'll land on the Admin Dashboard

### 7. Run
```bash
flutter run
```

---

## Build for Android Release
```bash
keytool -genkey -v -keystore ~/upload-keystore.jks -keyalg RSA -keysize 2048 -validity 10000 -alias upload
flutter build appbundle --release
# Output: build/app/outputs/bundle/release/app-release.aab
```

## Project Structure
```
lib/
├── main.dart
├── firebase_options.dart        ← generated by FlutterFire CLI
├── models/                      ← data models
├── services/                    ← Firebase logic
├── screens/
│   ├── customer/                ← home, builder, cart, orders, chat
│   └── admin/                   ← dashboard, inventory, orders, discounts, chat
└── widgets/                     ← reusable UI components
```
